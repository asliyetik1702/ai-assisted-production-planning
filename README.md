# Production Dashboard Project

## Project Status

This project is currently under development.

The aim is to build a production and inventory dashboard for a manufacturing company using real-world operational data. The project will focus on transforming raw and inconsistent datasets into a structured decision-support system.

## Project Objectives

- Consolidate production and inventory data from multiple sources
- Create a consistent SKU and product-name mapping
- Clean duplicate, missing and inconsistent records
- Monitor inventory levels and production activity
- Identify slow-moving, critical and high-value inventory items
- Develop meaningful production and inventory KPIs
- Build an interactive dashboard for management reporting
- Support operational and data-driven decision-making

## Planned Analysis

The project is expected to include:

- SKU standardization
- Product master-data creation
- Inventory consolidation
- ABC inventory classification
- Stock movement analysis
- Production quantity analysis
- Inventory turnover analysis
- Slow-moving and obsolete stock detection
- Demand pattern analysis
- Safety stock and reorder point calculations
- Capacity and production performance indicators

## Planned Dashboard Features

- Inventory overview
- Production overview
- KPI summary cards
- Product and SKU filters
- Stock status visualization
- ABC classification results
- Slow-moving inventory analysis
- Production trends
- Management-level summary views
